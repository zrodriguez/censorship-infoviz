### How to use the Censorship Dashboard
Set up and install dependencies:

1. pip3 install -r requirements.txt
2. python3 -m venv censorship-venv && source censorship-venv/bin/activate

Run:
1. python3 app.py

Close:
1. CLTR + C
2. source deactivate
